# Shop - MERN

### React/Redux
### node.js/express
### mongoDB

## For both the backend and frontend to run concurrently, in the root folder run:
```
npm run dev
```
## For just the server or client:
```
npm run server
```
```
npm run client
```
## Seeders

```
npm run data:import
```
